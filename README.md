# 🌦️ Live Weather Dashboard

A modern and interactive weather dashboard developed using **Python**, **Streamlit**, and the **OpenWeather API**. This application provides real-time weather information for any city worldwide through a clean and user-friendly interface.

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Current temperature
- 🤗 Feels like temperature
- 💧 Humidity
- 🌬️ Wind speed
- 📊 Atmospheric pressure
- 👀 Visibility
- 🌅 Sunrise & 🌇 Sunset timings
- 📍 Latitude & Longitude
- 📅 Current date and time
- ☁️ Weather condition with icon
- ❌ Error handling for invalid city names
- 🎨 Responsive and modern UI

## 🛠️ Technologies Used

- Python
- Streamlit
- Requests
- OpenWeather API

## ▶️ Installation

```bash
pip install streamlit requests
streamlit run app.py
```
